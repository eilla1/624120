# `@exu3`'s Prettier Configurations

## Install

```sh
yarn add -D @exu3/prettier-config
# or
npm install @exu3/prettier-config --save-dev
```

then add to `package.json`:

```json
{
  "prettier": "@exu3/prettier-config"
}
```
